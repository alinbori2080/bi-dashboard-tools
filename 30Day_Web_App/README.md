# 名将成长计划数据

本工具用于分析名将成长计划私域用户的 2-30 日留存，并与 GDATA 大盘留存对比。

工具是本地网页应用，双击 `start.cmd` 后会启动本地服务并自动打开看板。

## 访问地址

- 默认地址：`http://127.0.0.1:8767/`
- 启动方式：双击 `30Day_Web_App/start.cmd`
- 如果服务已经启动，再次双击 `start.cmd` 会直接打开网页，不会重复启动服务。

## 前端工程化

`frontend/` 是 30 日留存看板的新前端工程目录，采用 `Vue3 + TypeScript + Vite`。

开发和构建命令：

```text
cd 30Day_Web_App/frontend
npm install
npm run dev
npm run build
```

- `npm run dev`：启动前端开发服务，接口请求代理到 `http://127.0.0.1:8767`。
- `npm run build`：生成 `frontend/dist`，用于服务器部署。
- 后端启动时会优先读取 `frontend/dist/index.html`；如果没有构建产物，则自动回退到旧版 `static/`。
- 飞书密钥和表格配置只由后端读取，不写入前端工程。

## 报表

刷新后会生成本地文件：

- `output/30日留存结果.xlsx`

文件包含一张表：

- `每日结果`：同时包含日、周、月三类周期数据，并对比 GDATA 大盘留存。

## 日常使用

### 查看飞书结果表

1. 双击 `start.cmd`。
2. 页面默认数据来源为 `飞书结果表`。
3. 页面打开后读取飞书结果表。
4. 使用时间范围、对比维度和指标趋势查看看板。
5. 可点击 `导出全部` 导出当前数据来源的全部报表行。

飞书结果表读取失败时，页面会明确提示失败，不会自动回退到本地结果文件。

### 使用本地源文件重新计算

1. 将最新源文件放入 `30Day_Web_App/data`。
2. 点击 `刷新数据`。
3. 工具会读取本地源文件，重新生成 `output/30日留存结果.xlsx`。
4. 检查看板、趋势图和明细表。
5. 确认无误后，点击 `确认同步飞书`。

刷新只生成本地结果，不会自动写入飞书。只有点击 `确认同步飞书` 后才会写入飞书结果表。

### 查看本地结果文件

页面数据来源可切换为 `本地结果文件`。

- 该选择只影响当前页面展示读取哪份结果。
- 切换来源后立即读取对应结果。
- 切换来源不会自动刷新本地源数据。
- 切换来源不会自动同步飞书。
- 本地结果不存在时，页面提示 `请先刷新数据`。

## 源文件规则

本地刷新会读取 `data` 目录中最新的匹配文件。

- `名将杀 私域需求*.xlsx`：私域用户注册、登录和留存源数据。
- `新登*`：GDATA 大盘新登留存数据，支持 CSV/XLSX。

`新登*` 文件用于提供：

- 新登账号
- 大盘次留
- 大盘 3 日留
- 大盘 7 日留

## 统计口径

### 每日结果

按 `周期类型` 展示私域用户留存。

主要字段包括：

- 周期类型
- 统计周期
- 开始日期
- 结束日期
- 用户数
- 次留
- 3日留
- 4日留至30日留
- 新登账号
- 大盘次留、3日留、7日留
- 统计截止日
- 已统计最高留存日
- 7日留
- 14日留
- 21日留
- 30日留
- 新登账号
- 大盘次留
- 大盘3日留
- 大盘7日留
- 私域留存与大盘留存差值

未成熟、缺失或无法计算的字段保持为空，不补 0。实际计算结果为 0 时正常展示 0。

### 对比维度

按当前页面的 `对比维度` 展示。

- 日维度：按注册日期展示。
- 周维度：按自然周统计；当源数据只覆盖部分周时，按该周实际存在日期输出汇总。
- 月维度：按自然月统计；当源数据只覆盖部分月份时，按该月实际存在日期输出汇总。
- 页面只保留每日结果一个报表，通过日、周、月切换维度。

### 指标趋势和指标卡

指标区域会跟随当前选中的趋势指标变化。

例如选择 `7 日留存` 时，只展示：

- 加权 7 日留存
- 较上周变化
- 较上月变化
- 较大盘变化

留存率和相关变化统一使用 `pp` 表示。没有可比数据时显示 `—`，不补 0。

### 时间范围

页面支持内置时间范围筛选，包括：

- 全部
- 近 7 天
- 近 30 天
- 活动开启后：`2026-05-26` 至最新日期
- 自定义

当时间范围为 `全部` 时，不展示较上周、较上月和较大盘变化。

## 对比规则

- 当前指标值：按页面当前筛选后的日期范围计算。
- 较上周：当前筛选日期段整体向前平移 7 天，对比同指标。
- 较上月：当前筛选日期段整体向前平移 30 天，对比同指标；若参考区间早于现有数据，则只统计实际存在的日期。
- 活动开启后：不统计较上周变化；活动开启未满 60 天时，较上月固定使用 `2026-04-01` 至 `2026-04-26` 作为对照组；满 60 天后恢复正常向前平移 30 天。
- 较大盘：当前所选指标减去同期大盘同指标。
- 较大盘里的较上周、较上月，也按同样的动态范围计算。
- 月维度下，较上月按上一个自然月周期对比。

## 飞书同步

同步逻辑按 `周期类型 + 开始日期 + 结束日期` 匹配记录，并按指标字段增量新增或补充更新。

- 已成熟字段正常写入。
- 未成熟字段保持为空，不写成 0。
- 后续观察期成熟后，只补充新成熟字段。
- 本次未成熟、缺失或计算失败的字段，不覆盖飞书中已有有效值。
- 某些日期或字段异常时，只跳过受影响字段，不影响其他正常日期和指标。
- 同步结束后页面展示本次新增、更新、跳过的字段数量和跳过原因。

### 飞书配置

后端优先读取环境变量；未设置时再读取 `config/feishu_sync_config.json`。公开仓库不要提交真实配置文件。

支持的环境变量：

- `FEISHU_APP_ID`
- `FEISHU_APP_SECRET`
- `FEISHU_APP_TOKEN`
- `FEISHU_DAILY_TABLE_ID`
- `FEISHU_APP_URL`

## 页面功能

- 数据来源切换：`飞书结果表`、`本地结果文件`。
- 时间范围筛选：按当前报表过滤日期。
- 对比维度：日、周、月。
- 指标趋势：次留、3 日留、7 日留、14 日留、21 日留、30 日留等当前页面可选指标。
- 指标卡：跟随当前趋势指标联动展示。
- 趋势图：展示当前指标趋势。
- 明细表：默认按日期或周期降序。
- 导出 CSV：导出当前数据来源的全部报表行，不受页面筛选和排序影响。
- 飞书链接：可在右侧飞书同步区域点击 `打开` 进入对应飞书结果表。

## 目录说明

- `data/`：本地源数据目录。
- `output/`：本地刷新后生成 `30日留存结果.xlsx`。
- `config/`：飞书配置和同步状态。
- `frontend/`：Vue3 + TypeScript + Vite 前端工程目录。
- `frontend/dist/`：前端构建产物，部署时交给服务器使用。
- `static/`：网页前端文件。
- `runtime_state.json`：最近一次页面运行状态。
- `start.cmd`：本地启动脚本。

## 常见问题

### 页面提示飞书读取失败

检查飞书配置、网络、应用权限和结果表字段是否可用。读取失败时不会自动回退到本地结果文件。

### 本地结果文件不存在

点击 `刷新数据`，先生成 `output/30日留存结果.xlsx`。

### 双击 start.cmd 没反应

确认电脑已安装 Python，并且命令行中可执行：

```text
python --version
```
