# 名将杀周报数据

本工具用于查看和生成名将杀周报相关数据，包含两张报表：

- `新用户转化率`
- `活跃用户关注私域占比`

工具是本地网页应用，双击 `start.cmd` 后会启动本地服务并自动打开看板。

## 访问地址

- 默认地址：`http://127.0.0.1:8765/`
- 启动方式：双击 `GDATA_Web_App/start.cmd`
- 如果服务已经启动，再次双击 `start.cmd` 会直接打开网页，不会重复启动服务。

## 日常使用

### 查看飞书结果表

1. 双击 `start.cmd`。
2. 页面默认数据源为 `飞书结果表`。
3. 点击 `刷新数据`，从飞书结果表读取当前数据。
4. 使用时间范围、对比维度、指标切换查看看板。
5. 可点击 `导出 CSV` 导出当前页面筛选后的报表。

飞书结果表模式下，数据已经来自飞书，不需要也不能再次同步飞书。

### 使用本地源文件重新计算

1. 将最新源文件放入 `GDATA_Web_App/data`。
2. 页面数据源切换为 `本地文件`。
3. 点击 `刷新数据`。
4. 检查看板、趋势图和明细表。
5. 确认无误后，点击 `确认同步飞书`。

刷新只生成本地结果，不会自动写入飞书。只有点击 `确认同步飞书` 后才会写入飞书结果表。

## 源文件规则

本地刷新会读取 `data` 目录中最新的匹配文件。

- `新登*`：用于 GDATA 新登账号数据，必须包含 `日期`、`新登账号`。
- `企微分析_客户_趋势明细*`：用于 SCRM 新增客户数据，必须包含日期列和 `新增客户数`。
- `企微好友*`：用于私域关注占比数据，必须包含 `日期`、`新加好友`、`活跃账号`、`活跃累计好友`。

注意：`新登*` 文件需要在导出前筛选渠道：

- `官方渠道-全部`
- `CPS渠道-全部`
- `其他渠道-PC`

## 统计口径

### 新用户转化率

公式：

```text
新增客户数_SCRM / 新登账号_GDATA
```

页面展示字段包括：

- 周期类型
- 统计周期
- 开始日期
- 结束日期
- 新增客户数_SCRM
- 新登账号_GDATA
- 新用户转化率

### 活跃用户关注私域占比

公式：

```text
(活跃累计好友 - 新加好友) / (活跃账号 - 新加好友)
```

页面展示字段包括：

- 周期类型
- 统计周期
- 开始日期
- 结束日期
- 新加好友
- 活跃账号
- 活跃累计好友
- 活跃用户关注私域占比

### 周期规则

- 日维度：按自然日展示。
- 周维度：按周一至周日统计，只有完整自然周才生成周统计。
- 月维度：允许不完整自然月，但统计周期展示实际参与计算的开始日期和结束日期。
- 月维度下不统计 `较上周变化`。
- `较上月变化` 在月维度下按上一条月周期记录对比。

## 页面功能

- 数据源切换：`飞书结果表`、`本地文件`。
- 时间范围筛选：按页面内置时间范围过滤当前报表。
- 对比维度：支持日、周、月等当前页面可选维度。
- 指标卡：展示当前指标值、较上周变化、较上月变化等。
- 趋势图：展示当前指标趋势。
- 明细表：按当前筛选条件展示数据，默认按日期或周期降序。
- 导出 CSV：导出内容与当前页面数据源、筛选条件、排序和统计周期一致。
- 飞书链接：可在右侧飞书同步区域点击 `打开` 进入对应飞书结果表。

## 飞书同步

- 飞书结果表模式：只读飞书，不允许同步。
- 本地文件模式：刷新成功后可手动同步飞书。
- 同步前会检查是否存在计算错误。
- 同步成功后页面会展示新增、更新等结果。
- 同步失败时页面展示错误原因。

## 目录说明

- `data/`：本地源数据目录。
- `output/`：本地刷新后生成的 CSV 结果。
- `config/`：飞书配置和同步状态。
- `static/`：网页前端文件。
- `runtime_state.json`：最近一次页面运行状态。
- `start.cmd`：本地启动脚本。

## 常见问题

### 双击 start.cmd 没反应

确认电脑已安装 Python，并且命令行中可执行：

```text
python --version
```

### 页面显示飞书读取失败

检查飞书配置、网络、应用权限和结果表字段是否可用。

### 本地刷新失败

检查 `data` 目录下源文件是否齐全，字段名是否符合要求，日期是否能正常识别。
